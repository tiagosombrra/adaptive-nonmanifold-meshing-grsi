# Reproducibility protocol

The representative command builds the project in Release mode and executes the Eistute configuration used for the article. The configuration fixes one process, one thread and four adaptive stages.

The principal target is the stage-4 result: 7,184 triangles, normalized error 0.0367, 96% error reduction, 59% of triangles with quality at least 0.60, and zero mean shared-curve compatibility error.

Runtime depends on the compiler and hardware. The numerical targets, rather than wall-clock time, define successful reproduction.
